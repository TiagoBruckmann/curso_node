# curso_node

Desenvolvimento de uma API utilizando NODE.JS em conjunto com o Mongo DB

##### Pacotes utilizados

<ul>
    <li>mongoose</li>
    <li>express</li>
    <li>nodemon</li>
    <li>dotenv</li>
</ul>

### :gear: instalar dependencias

```sh
npm install
```

### :star: Executar aplicação

```sh
npm start
```
